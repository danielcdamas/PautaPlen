🚀 COMO USAR
═══════════════════════════════════════════════════════════════════════════════

1️⃣  INSTALAR (primeira vez)
    Windows:  setup.bat
    Linux:    bash setup.sh

2️⃣  TESTAR INSTALAÇÃO
    python test_pauta.py

3️⃣  USAR DE 3 FORMAS:

    a) CLI Interativa (fácil)
       python cli.py
       → Menu com opções

    b) Linha de comando (direto)
       python -m pauta_camara.main 10/06/26
       → Busca e baixa pauta

    c) VS Code Copilot (integrado)
       "Qual é a pauta de hoje?"
       "Traga a pauta de 15/06/26"
       → Usa automaticamente
